# Mern-expense-tracker
# MERN Stack Expense Tracker

This is a **full-stack Expense Tracker application** built using **React, Node.js, Express, and MongoDB**.  
Users can track **income and expenses**, and the **dashboard automatically calculates totals and balance**.  

---

## Features

- **JWT Authentication** – secure login
- **Add Expense / Income** – track transactions
- **Dashboard Calculation** – total income, total expenses, and balance
- **Expense List** – view all transactions dynamically
- **Full Stack Integration** – React frontend + Node.js/Express backend + MongoDB

---

## Tech Stack

| Frontend      | Backend         | Database       | Authentication |
| ------------- | --------------- | -------------- | -------------- |
| React.js      | Node.js         | MongoDB        | JWT            |
| Axios         | Express.js      | Mongoose ORM   |                |
